# Predicting Song Popularity Using Machine Learning

This project analyzes **26,000+ Spotify tracks** to predict song popularity using machine learning models based on audio and listener behavioral features.

## Dataset

The dataset contains audio features extracted from Spotify tracks, including:

- danceability
- energy
- loudness
- tempo
- valence
- acousticness
- speechiness

These features are commonly used to characterize musical structure and listener engagement patterns.

## Methods

The following machine learning models were implemented using **scikit-learn**:

- Logistic Regression
- Decision Tree Classifier

## Workflow

1. Data preprocessing and cleaning
2. Exploratory Data Analysis (EDA)
3. Feature engineering and correlation analysis
4. Model training and evaluation

## Evaluation

Model performance was evaluated using:

- Cross-validation
- ROC-AUC
- Classification accuracy

The models achieved approximately **78% accuracy** in predicting song popularity.

## Tools

- Python
- pandas
- scikit-learn
- matplotlib
- seaborn

## Objective

The goal of this project is to understand which **audio and behavioral signals** are most predictive of song popularity and demonstrate machine learning modeling workflows on real-world music data.
